# Vite-Gourmand #
Projet ECF - Développeur Web et Web Mobile  

Vite & Gourmand est une entreprise Bordelaise, spécialisée dans la préparation de plats, depuis 25 ans, pour tout type d'évènement. Elle est gouvernée par José et Julie. 
Aujourd'hui, dans une ère hyper-connectée, ils souhaitent évidemment développer leur application web et mobile afin de promouvoir les menus et permettre la commande aux visiteurs et ainsi faire croître leur image de marque.


*Objectifs de l'application*
- Présenter les menus disponibles
- Permettre aux clients de commander en ligne
- Gérer les commandes et les utilisateurs (en tant qu'administrateur)
- Générer des statistiques


*Stack technique*
- Front-end : HTML, CSS, JavaScript
- Back-end : PHP (PDO)
- Base SQL : MySQL
- Base NoSQL : MongoDB
- Déploiement : Fly.io / Vercel
